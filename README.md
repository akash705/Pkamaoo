## 🎯 Mission

PaisaKamaoo proactively reaches vulnerable Indian internet users through targeted social media advertising, simulating authentic "earn money from home" scam experiences to deliver empathetic, experiential cybersecurity education.

**The Problem:** In just 5 months of 2025, Indians lost ₹7,000+ crore to online scams. Traditional awareness campaigns are passive and ineffective.

**Our Solution:** Active, experiential education that meets users where scammers find them.

---

## 🚀 Key Features

### MVP (Current Development)
- **🎭 Authentic Scam Simulation** - Realistic "earn money from home" experience
- **⏰ Psychological Triggers** - 15-minute urgency timer and social proof
- **🎓 Empathetic Education** - "Congratulations! You just experienced how scammers target people like you"
- **🔄 Viral Sharing** - Built-in mechanism to test friends and family
- **🌐 Bilingual Support** - Hindi/English content for Indian demographics
- **📱 Mobile-First Design** - Optimized for social media traffic

### Coming Soon
- Multiple scam scenarios (romance, tech support, investment)
- Advanced analytics and sharing metrics
- Community features and user reports
- Mobile app for broader reach

## 🚦 Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn
- Git

### Installation

```bash
# Clone the repository
git clone https://github.com/akash/paisakamaoo.git
cd paisakamaoo

# Install dependencies
npm install

# Start development server
npm run dev

# Open http://localhost:3000
```

### Environment Setup

```bash
# Copy environment template
cp .env.example .env.local

# Configure required variables
NEXT_PUBLIC_SITE_URL=http://localhost:3000
NEXT_PUBLIC_ANALYTICS_ID=your_analytics_id
```

## 🎨 Design Principles

1. **Authentic Simulation** - Realistic scam replication using proven psychological triggers
2. **Empathetic Education** - Focus on empowerment, not shame
3. **Mobile-First** - Optimized for social media traffic patterns
4. **Cultural Sensitivity** - Hindi/English bilingual approach
5. **Viral by Design** - Built-in sharing mechanisms for organic growth

---

## 🔒 Ethical Considerations

- **No Data Storage** - MVP doesn't store personal information
- **Transparent Intent** - Clear educational purpose revelation
- **Empathetic Messaging** - Supportive, non-judgmental approach
- **Legal Compliance** - Adherence to Indian cybersecurity regulations
